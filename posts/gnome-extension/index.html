当时给大家推荐的20个扩展, 其中有相当一部分已经404 / 不再支持高版本的GNOME(47, 48, 49)了.

同时, 我看插件官网上的插件数量也从当时的60页涨到了73页:

话不多说, 我们开始吧!

按首字母排序

Copyous - 非常好用的剪切板管理器

  • 源代码: https://github.com/boerdereinar/copyous

  • GNOME扩展地址: https://extensions.gnome.org/extension/8834/copyous/

  • 特点: 设置支持 中文, GNOME原生, 可将剪切板内容快速生成二维码便于分享.

链接预览和图片预览似乎有点毛病. 除此之外体验不错

Custom Command Menu - 自定义指令集

  • 源代码: https://github.com/StorageB/custom-command-menu

  • GNOME扩展地址: https://extensions.gnome.org/extension/7024/custom-command-list/

  • 特点: 添加的指令支持 自定义图标, 支持 导入导出数据.

实际体验非常不错. 我添加了之前自己写的picup图床, 写博客效率更上一层楼

Do Not Disturb While Screen Sharing Or Recording Gnome Extension - 录屏时开启勿扰

  • 源代码: https://github.com/marcinjahn/gnome-do-not-disturb-while-screen-sharing-or-recording-extension

  • GNOME扩展地址: https://extensions.gnome.org/extension/5985/do-not-disturb-while-screen-sharing-or-recording/

  • 特点: 正如扩展名称, 在 录屏 或者 屏幕分享 时会自动开启勿扰模式, 此时不会有弹窗通知.

好用不多说, 每次装Linux必装扩展之一

Focus changer - 窗口焦点转移快捷键

  • 源代码: https://github.com/martinhjartmyr/gnome-shell-extension-focus-changer

  • GNOME扩展地址: https://extensions.gnome.org/extension/4627/focus-changer/

  • 特点: 同时开 多个 需要编辑内容的窗口, 但又 不喜欢用鼠标点击切换, 那么这款扩展相当适合你.

适合文字工作者 / 不喜欢用鼠标的人, 对照材料写文字时很方便

Lunar Calendar 农历 - 在日期 / 日历面板显示农历和节日

  • 源代码: https://gitlab.gnome.org/Nei/gnome-shell-extension-lunar-calendar/

  • GNOME扩展地址: https://extensions.gnome.org/extension/675/lunar-calendar/

  • 特点: 支持 农历, 干支, 节日.

我主要用来看日历面板中的农历和节日, 很方便

Media Controls - 顶栏中的媒体控制器

  • 源代码: https://github.com/cliffniff/media-controls

  • GNOME扩展地址: https://extensions.gnome.org/extension/4470/media-controls/

  • 特点: 支持 快进 / 快退, 上 / 下切歌, 暂停, 滚动显示歌曲信息, 播放模式切换.

美观, 方便, 必装. 此外还能控制浏览器中的视频播放

Right Click Next - 右键窗口移至下一桌面

  • 源代码: https://github.com/derVedro/gnome-right-click-next

  • GNOME扩展地址: https://extensions.gnome.org/extension/7600/right-click-next/

  • 特点: 在 概览界面 (类似安卓的后台, 苹果的多任务那个界面), 右键窗口 即可将其移动到 下一个桌面.

这应该是预装级别的扩展…

Screencst with Extra feature - 扩展GNOME自带的录屏(支持麦克风 / 屏幕声音)

  • 源代码: https://github.com/WSID/gnome-shell-screencast-extra-feature

  • GNOME扩展地址: https://extensions.gnome.org/extension/8764/screencast-extra-feature/

  • 特点: 将原本的录屏功能扩展至支持录制 屏幕声音 / 麦克风, 支持调整 帧率 与 分辨率.

这也应该是预装级别的扩展, 我直接卸载了OBS

SoundBar - 音频可视化

  • 源代码: https://gitlab.com/karthickk/gnome-shell-extension-soundbar

  • GNOME扩展地址: https://extensions.gnome.org/extension/8772/soundbar/

  • 特点: 在顶栏可视化 音频波浪图, 支持 自定义颜色, 高度 等.

好看, 精细

Veil - 收纳顶栏图标

  • 源代码: https://github.com/dagimg-dot/veil

  • GNOME扩展地址: https://extensions.gnome.org/extension/8786/veil/

  • 特点: 支持展开与收纳 动画, 自定义 图标, 自定义 收纳内容, 收纳方式(触碰 / 点击).

如果跟我一样安装的扩展多了, 用它做一个图标收纳, 顶栏又会干干净净

Volume Percent Display - 调节音量时显示百分比

  • 源代码: https://github.com/pic16f877ccs/Volume-percent-display

  • GNOME扩展地址: https://extensions.gnome.org/extension/8896/volume-percent-display/

  • 特点: 显示 百分比, 可搭配Volume Scroller使用.

其实用处不大

Weather O’Clock - 时间前 / 后的天气显示

  • 源代码: https://github.com/CleoMenezesJr/weather-oclock

  • GNOME扩展地址: https://extensions.gnome.org/extension/5470/weather-oclock/

  • 特点: 在顶栏时间的前 / 后 显示天气, 数据来自 gnome-weather, 因此可以自动定位.

看天气很方便

好啦! 本次的介绍就是这些, 一共 12个 全新的扩展, 都适配了高版本的GNOME, 喜欢可以试试看.

如果你有更好用的扩展, 记得在评论区跟大家分享~
